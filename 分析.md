#电影管理系统
服务端：提供API
客户端：ajax请求接口获取数据，使用数据渲染页面
npm init -y
tsc --init
服务端：ts+express+mongodb+class-validator+class-transformer
客户端：ts+react全家桶（react-router,redux,antd）
用react脚手架：npx -y create-react-app client --template typescript

tslint:跟esslint相似，是用于检查代码风格
npm install tslint --save-dev

//文件上传
content-type:multipart/form-data
服务器如何得到上传的文件 express中使用multer中间件
npm install --save multer
npm install --save @types/multer
